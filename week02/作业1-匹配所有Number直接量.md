# 正则表达式 匹配所有Number直接量

```js
(^0[bB][01]+$)|(^0[oO][0-7]+$)|(^0[xX][0-9a-fA-F]+$)|(^[1-9]\d+$)|(^([1-9]\d+|0)\.\d$)
```

## 二进制
```js
/^0[bB][01]+$/g.test('0b00100011');
```

## 八进制
```js
/^0[oO][0-7]+$/g.test('0o6443433');
```

## 十六进制
```js
/^0[xX][0-9a-fA-F]+$/g.test('0xa1bd12345');
```

## 十进制
### 整数
```js
/^[1-9]\d+$/g.test('123123');
```

### 浮点数
```js
/^([1-9]\d+|0)\.\d$/g.test('22342.24234');
```
